Curso-git UvigoMotorsport Esta es la pagina del curso de git de uvigo motorsport

Git es un software de control de versiones que nos permite registrar los cambios según quien y cuando se hacen. El conocimiento de git es muy valorado en el mercado ya que es clave para un buen trabajo colaborativo en proyectos de software.

Github es una plataforma que nos permite subir y descargar codigo controlado por git para trabajar en equipo de una manera controlada
